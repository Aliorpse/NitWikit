# 机器人框架
## Java
### NoneBot
NoneBot 是一个基于 Python 的机器人框架，支持同时接入多个平台，使用插件可实现搭建群服互通机器人。   
示例插件：[MCQQ](https://github.com/17TheWord/nonebot-plugin-mcqq)
### Koishi
Koishi 是一个跨平台、可扩展、高性能的聊天机器人框架,支持同时接入多个平台。使用插件可实现搭建群服互通机器人。 
示例插件：[minecraft-sync-msg](https://github.com/Twiyin0/koishi-plugin-minecraft-sync-msg)
### Yunzai & Karin
Yunzai 是一个基于 Node.js ，跨平台的~~原批~~机器人框架。使用插件可实现多种MC相关功能。
实例插件：[mctool-plugin](https://github.com/Dnyo666/mctool-plugin)
[mc-plugin](https://github.com/CikeyQi/mc-plugin)
## Bedrock
### SparkBridge
轻量级、模块化和多功能的Minecraft Bedrock Server BDS 服务器QQ机器人适配器，内置群服互通功能，支持插件接入，支持内核分离级别的二次开发。
https://sparkbridge.cn/
